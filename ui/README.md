# UI (Red/Green BattleBots-style)

Open directly in browser:

```bash
open /Users/tjb/.openclaw/workspace-scout/agent-arena/ui/index.html
```

Or serve locally:

```bash
cd /Users/tjb/.openclaw/workspace-scout/agent-arena/ui
python3 -m http.server 8787
# open http://localhost:8787
```

This is a fast screenshot/demo UI for receipts. It does not write onchain transactions.
